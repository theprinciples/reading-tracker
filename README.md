# reading-tracker
📚 네이버 프리미엄 콘텐츠, 책, 아티클 읽기 트래커
브라우저 하나로 동작하는 독립형 독서 관리 도구입니다.

CSV, Excel, Google Sheets에서 읽기 목록을 불러와 
진행률을 추적하고 출석 체크로 독서 습관을 시각화합니다.

⚠️ 모든 데이터는 브라우저 localStorage에 저장됩니다. 브라우저 기록 삭제 시 데이터도 함께 삭제됩니다.

---

## 🚀 시작하기

1. `index.html` 파일을 다운로드합니다
2. 브라우저에서 파일을 열면 바로 사용 가능합니다

---

## 📂 데이터 불러오기

- CSV / Excel (.xlsx, .xls) 파일을 드래그 앤 드롭하거나 클릭하여 업로드
- 컬럼 자동 감지 → 미리보기 → 매핑 확인 후 적용
- Google Sheets URL 붙여넣기 (시트를 "링크가 있는 모든 사용자에게 공개"로 설정)
- 수기 항목 추가 가능 (➕)
- Excel 여러 시트 자동 인식 → 각 시트를 별도 탭으로 불러오기

---

## ✅ 읽기 추적

- 체크박스 클릭으로 읽음/안읽음 관리
- 읽은 날짜 자동 기록
- 범위 체크: 시작~끝 번호 입력하여 일괄 체크/해제
- 전체 체크/해제 지원
- 제목 중복 자동 감지
- 잘린 제목은 팝업으로 표시

---

## 📊 대시보드

- 진행률 도넛 차트
- 월별 읽기 속도 (년도 필터 지원)
- 카테고리 분포 차트
- 검색 & 필터: 제목, 카테고리, URL 통합 검색. 카테고리별/읽음 상태별 필터, 정렬 지원

---

## 💡 출석 체크 & 전구

- 캘린더에서 읽은 날짜 클릭 → 출석 체크
- 체크할 때마다 전구에 빛이 채워짐
- 목표 설정: 30 / 66 / 100 / 200 / 365일
- 목표 달성 시 전구가 환하게 빛남
- 리셋 가능

---

## 🧠 마인드맵

- 아티클 편집(✏️)에서 태그 입력 (#키워드; 예시: #AI #코인 #구글 #버티브)
- 태그로 연결된 아티클이 노드 네트워크로 시각화
- 전체 / 그룹별 / 탭별 필터

---

## ⭐ 즐겨찾기

- 아티클 옆 ☆ 클릭으로 즐겨찾기 추가
- 즐겨찾기 탭에서 모든 탭의 별표 항목 통합 조회
- 탭별 필터 가능

---

## 🗒️ 노트

- URL 입력 시 🗒️ 클릭으로 링크 열기
- 텍스트 입력 시 💬에 팝업 표시
- URL + 텍스트 동시 입력 가능

---


## 🎨 테마 컬러

- 골드 / 블루 / 그린 / 퍼플 / 로즈

---

## 🔧 탭 관리

- 탭 추가/삭제/이름 변경
- 그룹 설정 (같은 그룹은 홈에서 함께 표시)
- 드래그 앤 드롭으로 탭 순서 변경
- 탭 네비게이션: ◀▶ 한 탭씩, ◀◀▶▶ 처음/끝으로 이동

---

## ⚙️ 설정 / 도움말

- 탭바 오른쪽 ⚙️ 아이콘으로 열기
- 전체 사용법 가이드 제공
- 🗑️ 전체 데이터 초기화: 모든 탭, 읽기 기록, 즐겨찾기, 출석 체크 삭제
